MAKING SNAKE

Game Panel

Here is the game panel part of the code. In total it is 181 lines of code, and this is what executes the entire game! It took around 1 hour straight to write this code.

Game Frame

This is a class with a construction that calls on the different classes in the game panel, that makes the game run!

Main

This is the main part of the code. You would think the main code is the biggest and most advanced, but they rarely are! You will learn that we want to make the main code as easy as possible. In this case, all it does is call on the game frame, which calls on the game panel!

Final product


Like we touched on in the Reflection page, having some fun while learning to code is important. It is important both for motivation and for learning. With your own side projects you can see what type of cool stuff programming can do, not just the boring curriculum (no offence).
